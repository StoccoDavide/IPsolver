Detailed Description

template<typename Real, Integer N = Eigen::Dynamic, Integer M = Eigen::Dynamic>
class IPsolver::ProblemWrapper< Real, N, M >

This class is a simple wrapper around the Problem class, allowing for easy instantiation and usage. It inherits from the Problem class and can be used to define specific optimization problems.

Template Parameters
RealThe floating-point type.
NThe size of the primal variable vector (default is dynamic).
MThe size of the dual variable vector (default is dynamic).
