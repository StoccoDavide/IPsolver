Detailed Description

template<typename Real, Integer N = Eigen::Dynamic, Integer M = Eigen::Dynamic>
class IPsolver::Problem< Real, N, M >

The Problem class serves as a base class for defining optimization problems in the IPsolver library. It provides a structure for encapsulating the problem's objective function, constraints, and other necessary components.

Template Parameters
RealThe floating-point type.
NThe size of the primal variable vector (default is dynamic).
MThe size of the dual variable vector (default is dynamic).
